# LumberLogger
A simple wood moisture logger

## Hardware
* Arduino Nano 33 IOT
* TI Half-Bridge L293D
* Resistors, ?x22M, ?x10M, ?x1M
* Breadboard and wires
* Screws(2) 8x35mm
